漏洞简介

什么是log4j2漏洞？

简单来说，就是构造一个 ${} 格式的payload，当log4j2接收到后，会根据 ${} 中的内容去调用相对应的lookup方法，从而解析执行payload。

影响范围：2.0-beta9 <= Apache Log4j <= 2.15.0-rc1（1.x不受影响）
