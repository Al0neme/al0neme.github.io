Thief - A light webshell not use session

应用场景

在渗透测试中，无论是Godzilla还是AntSward，在通信时都会涉及到session，本身是一个很好的姿势，但是，在有些实际情况中，会遇到测试目标根本不通过Session通信的情况，脚本解析时根本不下发session，所以就copy了大部分Godzilla中的代码，结合AI凑出了这个工具，这也是为啥叫Thief的原因

为什么不二开Godzilla?

理由同上，反编译时发现Godzilla的代码通信中很多地方都涉及到了session，如果改需要大改，代码能力有限，所以选择 script joker 最喜欢的python重新编写。

一些bug

本身就是AI + 想到啥敲啥凑出来的，可能bug多，比如反斜杠，比如管道符等等，但是用在前期信息探测勉强够用了，不想再折腾了（又不是不能用0.0 ....）

新的功能?

本工具初衷是为了不通过session进行webshell通信，只写了一些基本的功能，代码结构其实比较简单，你可以自己二次开发
